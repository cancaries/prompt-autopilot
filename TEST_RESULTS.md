## 问题记录

### Issue #1: 英文输入 → 中文模板字段（语言不一致）
- **影响用例**: T4, T7
- **现象**: 用户输入为英文，但输出模板的受众/策略等结构化字段使用中文
- **示例**:
  - T4: 输入 "write a blog post about AI"（英文）→ 输出 "受众画像：一般读者，对AI有基本了解"（中文）
  - T7: 输入 "explain how blockchain works"（英文）→ 输出 "受众画像：一般读者，对 blockchain 有基本了解"（中文）
- **严重程度**: 中（不影响功能运行，但用户体验割裂）
- **修复方向**: 英文输入 → 英文模板字段

### Issue #2: Few-shot示例与任务无关
- **影响用例**: T16
- **现象**: T16（写文献综述摘要）输出中 Few-shot 部分为空，未填充
- **严重程度**: 低（摘要类任务本身缺少天然 Few-shot 示例，可接受）
