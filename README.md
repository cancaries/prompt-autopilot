## ⚡ See It In Action

```
$ pma "帮我写排序算法"

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
✨ 优化后的编程指令
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

## 🎯 任务
用 Python 实现快速排序算法

## 📥 输入
类型：整数数组，范围：1-100000，示例：[3, 6, 8, 10, 1, 2, 1]

## 📤 输出
类型：整数数组（升序），示例：[1, 1, 2, 3, 6, 8, 10]

## ⚡ 性能
时间：O(n log n) | 空间：O(log n)

## 🛡️ 边界
空数组 → [] | 单元素 → [x] | 重复 → 保持相对顺序

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Score: 9.0/10
```
